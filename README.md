
  # Valentine's Day Proposal Page

  This is a code bundle for Valentine's Day Proposal Page. The original project is available at https://www.figma.com/design/9ne1vgdhlzcHdQqGbB9kw7/Valentine-s-Day-Proposal-Page.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

    ## Notes

    - The proposal images are now bundled locally from `src/assets` (no `figma:asset` imports).
